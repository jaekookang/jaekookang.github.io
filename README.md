## Note
- Local build
	- (1) Manually clean up `_site` directory (Don't use `bundle clean`)
	- (2) Build: `bundle exec jekyll build`
	- (3) Build & Run: `bundle exec jekyll liveserve`

- Updating procedure
	- (1) clean up `_site` directory 
	- (2) Build
	- (3) Run `. ./publish.sh "COMMIT MESSAGE"`
