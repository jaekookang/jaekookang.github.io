## Note
- Local build
	- Option 1. Build: `bundle exec jekyll build`
	- Option 2. Build & Run: `bundle exec jekyll liveserve`

- Updating procedure
	- (1) Build: `bundle exec jekyll build`
	- (2) Run: `. ./publish.sh "COMMIT MESSAGE"`
